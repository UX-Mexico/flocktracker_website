Downloading the Flocktracker Application
An early version of the application is now available for beta testers! To learn more, please make sure to sign up on our contacts list to receive future updates. The download of the application will be initialized by clicking the above icon. Before running the application, please be sure that you have read and understand the User's Manual, which elaborates on how to install and operate the application. Do note that this project is currently in progress, and the user's manual is constantly being updated and finalized, as with the application. Please keep on the lookout for future updates to the application and user's manual and feel free to email us if you have any further questions.
